# Transaction Commands

Implement transaction CLI commands: list and update.

## Location
`src/monarch_cli/commands/transactions.py`

## Commands

### `monarch transactions list`
List transactions with optional filters.

Parameters:
- `--limit, -l`: Max transactions to return (default 100)
- `--offset, -o`: Pagination offset (default 0)
- `--start, -s`: Start date (YYYY-MM-DD)
- `--end, -e`: End date (YYYY-MM-DD)
- `--preset, -p`: Date range preset (overridden by --start/--end)
- `--account, -a`: Filter by account ID
- `--search`: Search transactions
- `--format, -f`: Output format
- `--ndjson`: Stream as NDJSON
- `--raw`: Raw API response

### `monarch transactions update`
Update an existing transaction.

Parameters:
- Transaction ID (positional argument, required)
- `--amount`: New amount
- `--description, -d`: New description/merchant name
- `--category, -c`: New category ID
- `--dry-run`: Preview changes without applying

## CLI Examples
```bash
# List transactions
monarch transactions list
monarch transactions list --preset this-month
monarch transactions list --limit 50 --start 2024-12-01
monarch transactions list --account ACC123 --format table
monarch transactions list --search "Amazon"
monarch transactions list --ndjson | head -20

# Filter with jq
monarch transactions list | jq '[.[] | select(.amount < -100)]'
monarch transactions list | jq 'group_by(.category) | map({category: .[0].category, total: map(.amount) | add})'

# Update transactions
monarch transactions update TXN123 --category CAT456
monarch transactions update TXN123 --amount -50.00 --dry-run
monarch transactions update TXN123 --description "New Description"
```

```json

    "operation": "update_transaction",
    "transaction_id": "TXN123",
    "changes": {"category_id": "CAT456"},
    "message": "No changes made (dry run)"
}
```
