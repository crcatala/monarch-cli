# Live Tests Infrastructure (Local Dev Only)

Set up live tests that run against the real Monarch Money API (local development only).

## Purpose
Live tests verify that our code works with the actual Monarch Money API. They should NOT run in CI because:
- Require real credentials
- API responses vary per user
- Could hit rate limits
- API compatibility is the library's responsibility

## Test Location
```
tests/
└── live/
    ├── __init__.py
    └── test_live_api.py
```

## Implementation
```python
# tests/live/test_live_api.py
import os
import pytest

# Mark all tests in this file as live tests
pytestmark = pytest.mark.live

LIVE_ENABLED = os.environ.get("MONARCH_LIVE_TESTS") == "1"


@pytest.mark.skipif(not LIVE_ENABLED, reason="Live tests disabled (set MONARCH_LIVE_TESTS=1)")
class TestLiveAPI:
    """Tests against real Monarch Money API.
    
    ⚠️  LOCAL DEVELOPMENT ONLY - Do not run in CI!

    Prerequisites:
    - Authenticated via `monarch auth login`, OR
    - MONARCH_TOKEN environment variable set
    
    Run with:
        MONARCH_LIVE_TESTS=1 uv run pytest tests/live/ -m live
    """

    def test_get_accounts_returns_data(self):
        """Should fetch real accounts."""
        from monarch_cli.core.adapter import get_authenticated_client
        from monarch_cli.core.async_utils import run_async

        client = get_authenticated_client()
        accounts = run_async(client.get_accounts())

        assert "accounts" in accounts
        # Don't assert specific data - it varies per user

    def test_get_transactions_returns_data(self):
        """Should fetch real transactions."""
        from monarch_cli.core.adapter import get_authenticated_client
        from monarch_cli.core.async_utils import run_async

        client = get_authenticated_client()
        txns = run_async(client.get_transactions(limit=10))

        assert "allTransactions" in txns

    def test_get_categories_returns_data(self):
        """Should fetch real categories."""
        from monarch_cli.core.adapter import get_authenticated_client
        from monarch_cli.core.async_utils import run_async

        client = get_authenticated_client()
        cats = run_async(client.get_transaction_categories())

        assert "categories" in cats
        # Categories should have some items
        assert len(cats["categories"]) > 0
```

## pytest Configuration
Already in pyproject.toml:
```toml
[tool.pytest.ini_options]
markers = [
    "live: marks tests that call real APIs (deselect with '-m \"not live\"')",
]
```

## Running Live Tests
```bash
# Enable and run live tests
MONARCH_LIVE_TESTS=1 uv run pytest tests/live/ -m live -v

# Or add to Makefile
make test-live  # Runs: MONARCH_LIVE_TESTS=1 uv run pytest tests/live/ -m live
```

## CI Configuration
The CI workflow already excludes live tests:
```yaml
- name: Test (excluding live tests)
  run: uv run pytest -m "not live"
```

## When to Use Live Tests
- After implementing new API integrations
- When debugging unexpected API behavior
- Verifying transforms match actual data structure
- Manual verification before releases
