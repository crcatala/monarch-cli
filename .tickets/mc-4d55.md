### 5. AI Agent Integration
```markdown
## AI Agent Integration

Monarch CLI is designed for AI agent consumption. Key features:

### Structured Output
All commands output JSON by default for easy parsing.

### Error Codes
Errors include machine-readable codes:
```json
{"error": true, "code": "AUTH_REQUIRED", "message": "..."}
```

### Quiet Mode
Get just IDs for efficient processing:
```bash
monarch accounts list --quiet
```

### Example: Claude Tool Use
```python
tools = [{
    "name": "get_accounts",
    "command": "monarch accounts list"
}]
```
```
