# Main Entry Point (Auth Only)

Create the main CLI entry point with Typer app, initially with only auth commands.

## Location
`src/monarch_cli/main.py`

## Implementation
```python
import typer

app = typer.Typer(
    name="monarch",
    help="CLI for Monarch Money - AI agent friendly financial data access",
    no_args_is_help=True,
)

# Phase 1: Only auth commands registered
from .commands import auth
app.add_typer(auth.app, name="auth")

# Phase 3 will add: accounts, transactions, budgets, cashflow, categories

if __name__ == "__main__":
    app()
```

## Typer Configuration
- **name**: "monarch" - the command name
- **no_args_is_help=True**: Running just `monarch` shows help (user-friendly)
- **help**: Main description shown in --help

## Entry Point in pyproject.toml
Already configured in Phase 0:
```toml
[project.scripts]
monarch = "monarch_cli.main:app"
```

## Why Typer?
- Automatic help generation from function signatures
- Built-in completion support
- Rich integration for colors/formatting
- Type hints become CLI arguments
- Industry standard for modern Python CLIs

## Global Options (Future)
Phase 3 will add global options via callback:
```python
@app.callback()
def main(
    verbose: bool = typer.Option(False, "--verbose", "-v"),
    format: OutputFormat = typer.Option(OutputFormat.JSON, "--format", "-f"),
):
    set_verbose(verbose)
```

For Phase 1, we keep it simple - just the auth subcommand.
