# Cashflow Commands

Implement cashflow CLI command: summary.

## Location
`src/monarch_cli/commands/cashflow.py`

## Command

### `monarch cashflow summary`
Get income/expense analysis for date range.

```python
@app.command("summary")
@handle_errors
def cashflow_summary(
    start_date: Optional[str] = typer.Option(None, "--start", "-s"),
    end_date: Optional[str] = typer.Option(None, "--end", "-e"),
    preset: Optional[DatePreset] = typer.Option(None, "--preset", "-p"),
    format: OutputFormat = typer.Option(OutputFormat.JSON, "--format", "-f"),
):
    """Get income/expense analysis for date range."""
    resolved_start, resolved_end = parse_date_range(preset, start_date, end_date)

    with spinner("Calculating cashflow..."):
        client = get_authenticated_client()
        cashflow = run_async(client.get_cashflow_summary(
            start_date=resolved_start,
            end_date=resolved_end,
        ))

    output(cashflow, format)
```

## Date Range Options
| Option | Description |
|--------|-------------|
| --start, -s | Start date (YYYY-MM-DD) |
| --end, -e | End date (YYYY-MM-DD) |
| --preset, -p | Date preset (this-month, last-30-days, ytd, etc.) |

Explicit dates override preset.

## CLI Examples
```bash
monarch cashflow summary                           # Current period
monarch cashflow summary --preset this-month       # This month
monarch cashflow summary --preset last-30-days     # Last 30 days
monarch cashflow summary --start 2024-10-01 --end 2024-12-31
monarch cashflow summary --preset ytd -f table     # Year to date, table format
```

## API Response
The `get_cashflow_summary()` response includes:
- sumIncome: Total income
- sumExpense: Total expenses
- savings: Net savings
- savingsRate: Percentage saved
- byCategory: Breakdown by category

For MVP, we pass through the API response directly. A transformer can be added if needed for schema stability.

## Note on Detailed Cashflow
The `get_cashflow()` method provides more detailed breakdown by category/merchant. This is deferred to Priority 2 (`monarch cashflow detailed`).
