# Shell Completions Documentation

Document shell completions setup for Bash, Zsh, and Fish.

## Background
Typer has built-in shell completion support via `typer-cli` or the `--install-completion` flag. No code changes needed - just documentation.

## Documentation to Add to README

### Shell Completions Section
```markdown
## Shell Completions

Enable tab completion for your shell:

### Bash
```bash
monarch --install-completion bash
source ~/.bashrc
```

### Zsh
```bash
monarch --install-completion zsh
source ~/.zshrc
```

### Fish
```bash
monarch --install-completion fish
```

### Usage
After installation, use Tab to complete:
- `monarch tr<TAB>` → `monarch transactions`
- `monarch transactions li<TAB>` → `monarch transactions list`
- `monarch transactions list --<TAB>` → shows all options
```

## How Typer Completions Work
Typer generates completion scripts that:
1. List available subcommands
2. List available options for each command
3. Complete option values (for choices like --format)

## Verification
```bash
# Check completion is installed
monarch --show-completion bash
monarch --show-completion zsh
monarch --show-completion fish

# Verify completion works
monarch <TAB><TAB>  # Should show: auth, accounts, transactions, etc.
```

## Note
Some shells may require restarting the terminal for completions to take effect. The completion scripts are installed to:
- Bash: ~/.bashrc or ~/.bash_completion
- Zsh: ~/.zshrc (via compinit)
- Fish: ~/.config/fish/completions/
